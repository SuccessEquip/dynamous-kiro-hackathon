# Product Overview

## Product Purpose
CORE Framework is an interactive project planning tool that transforms vague ideas into actionable project plans through a structured 4-phase, 15-question methodology. It guides users through Clarify → Organize → Refine → Equip phases and outputs comprehensive documentation in multiple formats (Markdown, AI prompts, JSON) with session persistence and optional AI assistance.

## Target Users
**Primary Audiences:**
- **Solo entrepreneurs** - Need quick idea validation and structured planning
- **Developers** - Want systematic technical planning and requirements gathering
- **Small product teams (2-10 people)** - Require PRD-like documents and shared project clarity
- **Non-technical founders** - Need to communicate clear requirements to development teams

**Core Needs:**
- Structured requirements gathering and documentation
- Clear prioritization (MVP vs nice-to-have features)
- Risk analysis and mitigation planning
- Implementation-ready documentation for AI handoff
- Low-friction setup and ease of use

## Key Features
- **4-Phase Methodology**: Clarify → Organize → Refine → Equip workflow
- **15 Strategic Questions**: Comprehensive project analysis framework
- **Multi-Format Output**: Markdown documentation, AI prompts, structured JSON
- **Session Persistence**: Save and resume planning sessions
- **Progressive Complexity**: Three implementation tiers (HTML/JS, Python TUI, React/Supabase)
- **AI Integration**: Optional OpenRouter AI assistance for enhanced planning (v2/v3)
- **Accessibility First**: Full keyboard navigation and screen reader support
- **Cross-Platform**: Works on web, desktop, and mobile environments

## Business Objectives
- **User Adoption**: Enable rapid project planning for technical and non-technical users
- **Documentation Quality**: Generate implementation-ready project specifications
- **Workflow Efficiency**: Reduce time from idea to actionable plan by 70%
- **Team Alignment**: Improve project clarity and reduce miscommunication
- **Scalability**: Support individual users to small teams seamlessly

## User Journey
1. **Entry**: User arrives with vague project idea or concept
2. **Clarify Phase**: Answer foundational questions about project purpose and scope
3. **Organize Phase**: Structure requirements, features, and priorities
4. **Refine Phase**: Analyze risks, constraints, and success metrics
5. **Equip Phase**: Generate documentation and implementation roadmap
6. **Output**: Receive comprehensive project plan in preferred format
7. **Iteration**: Save session and refine plan over time

## Success Criteria
- **Completion Rate**: >80% of users complete all 4 phases
- **Documentation Quality**: Generated plans contain actionable, specific requirements
- **User Satisfaction**: Users report increased project clarity and confidence
- **Implementation Success**: Projects planned with CORE show higher completion rates
- **Adoption Growth**: Steady increase in session saves and repeat usage
- **Team Collaboration**: Multi-user sessions show improved alignment metrics
